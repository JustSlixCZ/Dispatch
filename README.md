Installation
Download the latest version.

Extract the downloaded .zip file into your server's resources folder.

Rename the folder to tnrp_dispatch (if it has a different name).

Add ensure tnrp_dispatch to your server.cfg file.

Configure the script by editing the config.lua file to match your server's settings (e.g., job names, colors, and UI preferences).

Restart your server or type refresh and ensure tnrp_dispatch in your server console.

Dependencies:
ox_lib
esx_framework
